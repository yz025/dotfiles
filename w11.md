# 5. ddf
```
cd ~/Documents/projects
git clone https://github.com/yz-5555/ddf.git
cd ddf
cargo build --release

cd ~/Documents
git clone https://github.com/yz-5555/dotfiles.git

$env:DDF_TARGET = "$HOME\Documents\dotfiles"
$env:Path += ";$HOME\Documents\projects\ddf\target\release"

ddf pull w11_desktop
```
